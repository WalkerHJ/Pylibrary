##  py头脑王者自动答题<br/> ##
最近比较火的头脑王者 冲顶大会 <br>
#### 思路:<br>
* 1:利用adb截图到本地保存到内存中 <br>
* 2:从内存中取出,并切取所需题目及答案部分并拼接到一张图<br>
* 3:文字识别
* 4:百度及谷歌搜索匹配最高的答案
* 5:adb模拟点击正确答案<br>
